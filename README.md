# ServiceNow Administrator Labs & Training Projects  

This repository documents my hands-on practice labs completed during a **ServiceNow Administrator Training Program**.  
The labs cover administration, configuration, and ITSM processes, demonstrating practical application of ServiceNow skills aligned with ITIL best practices.  

---

## üìö Modules & Labs  

### Module 1: Introduction to ServiceNow  
**Learned:** ITIL concepts, ITSM processes, and ServiceNow architecture.  
**Labs Completed:**  
- Explored core ITSM applications (Incident, Problem, Change).  
- Compared ServiceNow roles (Admin, End-User, ITIL user).  

---

### Module 2: Navigation and User Interface  
**Learned:** Efficient navigation using modules, lists, and forms.  
**Labs Completed:**  
- Customized Application Navigator with favorites.  
- Applied filters and created list views for incidents.  

---

### Module 3: User Administration  
**Learned:** User, group, and role management with ACLs.  
**Labs Completed:**  
- Created two users (Admin & ITIL role).  
- Built a new group and assigned it to a role.  
- Tested access permissions by impersonating users.  

---

### Module 4: Tables, Forms & Fields  
**Learned:** Data modeling and form customization.  
**Labs Completed:**  
- Created a custom table with dictionary entries.  
- Added new fields with different data types (string, choice, reference).  
- Modified form layouts and views.  

---

### Module 5: Service Catalog Management  
**Learned:** Building catalog items and record producers.  
**Labs Completed:**  
- Built a catalog item with variables and variable sets.  
- Configured a record producer to create incidents.  
- Linked catalog item requests to approval workflows.  

---

### Module 6: Workflow & Flow Designer  
**Learned:** Automating processes with workflows and flows.  
**Labs Completed:**  
- Designed a workflow for catalog request approvals.  
- Used Flow Designer to automate notifications for new incidents.  
- Tested flows by submitting requests as different users.  

---

### Module 7: Notifications and Events  
**Learned:** Communication through events and notification templates.  
**Labs Completed:**  
- Created a custom email notification for high-priority incidents.  
- Configured event triggers and tested message delivery.  

---

### Module 8: Reports and Dashboards  
**Learned:** Reporting and performance analytics.  
**Labs Completed:**  
- Built incident trend reports by priority.  
- Designed a dashboard displaying open vs. resolved tickets.  
- Shared dashboard with ITIL users.  

---

### Module 9: Import Sets and Data Management  
**Learned:** Data loading and transformation best practices.  
**Labs Completed:**  
- Imported external user data using Import Sets.  
- Created a Transform Map to map fields.  
- Applied Data Policies to enforce mandatory fields.  

---

### Module 10: Update Sets and Application Deployment  
**Learned:** Migrating customizations across instances.  
**Labs Completed:**  
- Created an Update Set for catalog item changes.  
- Tested moving configurations between dev and test instances.  

---

### Module 11: Instance Management and Configuration  
**Learned:** Instance cloning, system configuration, and plugin management.  
**Labs Completed:**  
- Configured system properties for notifications.  
- Explored plugin activation in a sub-production instance.  

---

### Module 12: Final Project & Best Practices  
**Learned:** End-to-end ServiceNow administration.  
**Labs Completed:**  
- Built a custom end-to-end catalog request process with approvals.  
- Automated notifications and dashboards for request tracking.  
- Applied performance tuning tips for faster load times.
